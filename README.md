## 环境配置

* 安装Dart SDK 和 Flutter SDK
* Android Studio 环境（包括Android SDK Gradle管理工具）
* 后端服务器的网络请求，``` base_url``` 在 ``` /monotes/lib/common/dio_utils``` 中设置，当后端服务器地址改变，更改```base_url```

## 使用说明

* 第一次登录时，需要注册信息。如果使用短信二维码验证，需要确保后端的短信发送功能正常工作。
